이미지태그

이미지태그는 단일 태그이며 속성으로 모든것을 설정한다. src=" " : 이미지에 대한 경로(local, 외부(http)) width=" " : 넓이(px) height=" " : 길이(px) : width, height 생략 시 자동으로 px단위로 적용되며 이미지 원본 크기로 출력된다. alt=" " : 이미지경로에 이미지가 없는 경우 출력될 문구를 지정
