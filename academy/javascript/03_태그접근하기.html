태그에 접근하는 방법

document객체의 메소드를 이용해서 접근을 할 수 있다. js에서는 페이지자체를 하나의 객체로 보고 페이지 내부의 태그들도 모두 단일 객체로 본다(이를 'DOM'이라고 함) * 객체는 key:value형식으로 되어있다. 태그에 접근하는 방법은 기본 4가지가 있다.

  • id값을 기준으로 태그 한개를 가져오는 방법 : getElementById
  • class값을 기준으로 다수태그를 가져오는 방법 : getElementByClassName()
  • tagName값을 기준으로 다수태그를 가져오는 방법 : getElementByTagName()
  • name값을 기준으로 다수태그를 가져오는 방법 : getElementsByName()

다수 태그를 가져오는 메소드는 배열방식(List)의 객체를 반환 한개 태그를 가져오는 메소드는 태그(Element객체)를 반환 함

id값을 기준으로 태그 가져오기

태그의 id속성의 값을 기준으로 한개의 태그만 가져옴 document.getElementById("id값"); -> html태그중 id값이 인수값과 같은 태그를 가져옴

class속성을 기준으로 태그를 가져오기

태그의 class속성에 있는 값을 기준으로 태그를 객체로 가져오는 함수 태그의 class속성은 페이지에서 중복이 가능하다. -> 여러값을 가져올 수 있다. 여러 태그객체가 반환됨(자바의 List객체랑 비슷한 방식의 객체가 반환됨) document.getElementsByClassName("class속성값");

CSS선택자로 태그를 불러오는 방법

css에서 사용하는 선택자를 이용해서 태그를 지정하여 가져오는 것 #test -> id속성이 test인 태그 .test -> class속성이 test인 태그 [type=text] -> type속성이 text인 태그 ul>li -> ul에 직접 연결된 li태그
document.querySelector("선택자") -> 한개의 태그만 가져오는 것, 다수가 있으면 제일 위에 선언된 태그만 가져옴. (첫번째 태그)
document.querySelectorAll("선택자") -> 한개이상의 태그를 가져오는 것
