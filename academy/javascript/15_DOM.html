document객체를 이용해서 태그를 동적으로 생성

document객체의 createElement()메소드를 이용해서 생성 -> 요소노드 예) p,img,audio 만일 텍스트 노드가 필요한 태그면 텍스트도 생성해줘야 한다. document객체의 createTextNode("문구")메소드를 이용 -> 텍스트노드 텍스트가 필요없는 태그면 생성된 요소노드에 속성만 설정해준다.


텍스트가 없는 단일 태그 생성하기

img, audio, video, input태그 단일태그들은 속성을 설정해야한다. -> document객체의 setAttribute()메소드를 이용해서 설정할 수 있다. .setAttribute(속성명, 대입값)

생성된 태그를 원하는 위치에 넣기

insertAdjacentElement("예약어(속성값",생성된태그);
속성값
beforebegin : 지정태그의 앞에(같은 레벨) afterend : 지정태그의 위에(같은 레벨) beforeend : 지정태그의 맨밑자식(appendChild와 동일) afterbegin : 지정태그의 맨위자식

기준태그

생성된태그 style속성 설정하기

생성된 태그객체에 style property를 이용해서 태그의 css속성을 변경할 수 있다. Element객체의 property는 직접접근이 가능함. 1. Element.style.css속성명=속성값 2. setAttribute("style",";;;");

* css속성에는 - 가 있으나 js에서는 다르게 입력된다. : background-color -> backgroundColor, font-size -> fontSize




특정태그에서 원하는 태그 검색하기(순회 탐색)

특정태그를 가져와 그 태그가 가지고 있는 자식태그, 형제태그, 부모태그, 확인하는 Element속성 children속성 : 지정된 태그가 가지고 있는 자식태그들 parentElement속성 : 지정된 태그의 바로위 부모태그 previousElementSibling속성 : 지정된 태그의 바로 위 태그(같은레벨) nextElementSibling속성 : 지정된 태그의 바로 아래 태그(같은레벨) * 여러번 이어서 사용이 가능
