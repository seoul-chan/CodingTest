데이터 출력하기

스크립트에서 데이터를 출력하는 방식을 두개로 나눌수 있음
1. 페이지(화면)에 출력하는 방법 2. 브라우저에 출력하는 방법(알림창, console탭)

  • 페이지 출력
    1. document.write("출력할구문||출력할 태그+출력구문")
    2. 태그(element).innerHTML="출력할구문||출력할 태그+출력구문"
  • 브라우저 출력
    1. 알림창출력 : [window.]alert("출력문구")
    2. console탭 출력 : [window.]console.log("출력할문구||변수")

페이지출력하기

document.write()함수이용해서 출력하기

innerHTML속성을 이용해서 값 변경하기

자바스크립트에서 태그는 element객체로 처리한다. 태그마다 innerHTML속성을 가지고 있음. innerHTML속성에 있는 값을 태그에 문구로 출력됨. 사용형식 태그선택.innerHTML="문구";

브라우저에 출력하기

알림창으로 출력하기 : alert()함수를 이용

console창에 출력하기 : console.log()함수를 이용

데이터 입력받기

데이터를 입력받는 방법은 두가지로 나눌 수 있음 1. 함수를 이용해서 입력 받는 방법
prompt(), confirm() 2. input태그를 이용해서 입력 받는 방법

함수를 이용해서 데이터 받기

confirm()함수 이용하기

confirm함수는 true/false를 반환해주는 함수 확인/취소 버튼이 있는 알림창이 뜨고 확인을 누르면 true를 반환, 취소를 누르면 false를 반환하는 함수 매개변수에는 알림창에 출력되는 문구를 넣는다.

prompt함수 이용하기

클라이언트에게 문구를 입력 받아 반환해주는 함수

input태그로 데이터 입력받기

input태그도 태그로 element객체로 처리함. input태그 property중 value가 있는데 input값을 입력하면 value property에 값이 저장됨.
