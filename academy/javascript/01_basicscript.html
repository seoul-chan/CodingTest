자바스크립트 선언하기

자바스크립트는 script태그 안에 작성하여 이용을 함 script태그는 선언된 위치에 상관없이 html페이지 안에 있으면 된다. script태그는 여러 개 선언해도 상관이 없다. 단 위에서 순차적으로 실행한다.

자바 스크립트 이용하는 방식

자바스크립트를 이용하는 방식은 3가지 방식이 있음

  1. inline : 태그 내부의 이벤트 속성에 직접 javascript코드를 작성하는 것 * 간단한 구문만 사용
  2. internal : html페이지 내부의 script태그를 선언한고 script태그 안에 javascript코드를 작성하는 것
  3. external : xxx.js파일을 만들어 javascript코드를 작성하고 html페이지에서 script태그 scr속성으로 그 파일을 불러와 적용하는 것 * 내부로컬파일 / 외부 http통신 방식(url주소)으로 불러올 수도 있음
