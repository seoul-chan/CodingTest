태그에 대한 영역크기 설정

width / height 구역은 세개의 구역으로 분할(border, padding, content)된다. * 기본적으로 크기영역(width,height)은 content영역만 해당 크기설정은 절대값, 상대값으로 설정 단위 : px(절대값) / %(상대값)

절대크기 : 브라우저 창의 크기가 줄어도 고정된 크기를 유지한다.

상대크기 : 브라우저 창의 크기가 줄어들면 같이 줄어든다.

태그별 배치속성

태그별로 block속성과 inline속성으로 구분할 수 있음. block : 한 라인 전체를 차지함 -> 개행이 됨. width height 조정 가능 inline : 라인 안에서 여러 태그를 표현가능 -> 개행되지않음, width height 조정 불가능, 크기는 라인에서 지정하는대로 처리함 기본태그의 배치속성을 변경할 수 있음 -> display속성을 이용
