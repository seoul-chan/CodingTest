색상

css로 색상을 표시할때는 색상영문명, 16진수RGB값, 0~255까지 RGB값 RGBA(색투명도결정), HSL/HSLA

background-color

태그마다 페이지에서 차지하는 각 영역이 있는데 그영역에 특정 색상을 표현할 수 있게 하는 속성

  • html
  • css
  • javascript
  • jquery

background-clip

태그(element)가 차지하고 있는 영역에 대한 범위설정 속성 border-box : border의 영역까지 배경으로 설정 padding-box : padding의 영역까지 배경으로 설정 content-box : content의 영역까지 배경으로 설정

border 영역
padding 영역
content 영역

background-image

태그 영역에 배경으로 사진을 삽입하는 속성 기본적으로 사진의 크기와 영역의 크기비율대로 바둑판식으로 배치됨.

background-repeat

바둑판배경을 변경해주는 속성 repeat(default), repeat-x/repeat-y, no-repeat

background-size

배경이미지의 크기를 조정할 수 있는 속성 contain : 이미지가 영역안에 모두 들어갈 수 있게 확대, 축소 -> 이미지가 짤림없이 모두 들어감. cover : 이미지로 영역을 모두 꽉차게 설정이 되게 확대, 축소하는 것 -> 일부만 출력될 수 있음 숫자/백분율 :

background-position

배경이미지의 위치를 조정하는 속성 left|right|top|bottom|center||숫자

background-attachment

페이지가 스크롤에 의해 움직일때 배경의 위치를 설정하는 속성 scroll : 페이지가 움직이면 따라서 움직으는 속성 -> 페이지 이동하면 안보임 fixed : 페이지가 움직여도 설정한 위치에 있는 속성 -> 페이지가 이동해도 계속 보임

linear-gradient

선형 그라데이션 속성 설정 background:liner-gradient(각도||방향,시작색,끝색);
