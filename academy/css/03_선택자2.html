문자선택자

  • ::first-letter : 첫번째 한글자만 style적용하는 것
  • ::first-line : 첫번째 줄에 style적용하는 것(스크린 출력 기준)
  • ::after : 태그의 끝문자에 문자를 추가
  • ::before : 태그의 시작에 문자를 추가
  • ::selection : 마우스가 드래그한 문자에 style적용

first-letter : 첫번째 한글자만 style적용하는 것
first-line : 첫번째 줄에 style적용하는 것(스크린 출력 기준)

after : 태그의 끝문자에 문자를 추가
before : 태그의 시작에 문자를 추가
selection : 마우스가 드래그한 문자에 style적용

구조선택자
